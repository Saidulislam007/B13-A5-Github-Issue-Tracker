1.What is the difference between var, let, and const?
answer-
#var-
1.Old way to declare variable,
2.Function scoped,
3.Reassign and redeclare করা যায়.

#let-
1.Modern JavaScript variable,
2.Block scoped ({ } এর ভিতরে কাজ করে),
3.Reassign করা যায়, কিন্তু redeclare করা যায় না.

#const-
1.Constant variable,
2.Block scoped,
3.Reassign বা redeclare করা যায় না.

2.What is the spread operator (...)?
answer-
Spread Operator (...)
Spread operator ব্যবহার করা হয় array বা object এর ভেতরের সব value আলাদা করে বের করার জন্য।


3.What is the difference between map(), filter(), and forEach()?
answer-
#map()
1.Array এর প্রতিটি element এর উপর কাজ করে |
2.নতুন array return করে |

#filter()
1.Condition অনুযায়ী element select করে|
2.নতুন array return করে|

#forEach()
1.শুধু loop চালায়|
2.কোনো নতুন array return করে না|

4.What is an arrow function?
answer- Arrow function (=>) ব্যবহার করে ছোট ও সহজভাবে function লেখা যায়।

5.What are template literals?
answer- Template literals ব্যবহার করে backtick (``) দিয়ে string লিখে তার মধ্যে variable বা expression ব্যবহার করা যায়।
